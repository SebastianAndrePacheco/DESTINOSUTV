# UTV's en Huacachina + Sandboarding — Destinos ICA

Landing page para el paseo en UTV + sandboarding en Huacachina (Ica, Perú).
Incluye un formulario que arma el pedido y lo envía por **WhatsApp** al **+51 955946137**.

## Cómo verla

Abre `index.html` en el navegador (doble clic). No necesita servidor.

Publicada con GitHub Pages en la pestaña **Settings → Pages** del repositorio.

## Contenido

- `index.html` — la página completa (HTML + CSS + JS en un archivo).
- `assets/` — imágenes optimizadas y fuentes.

## Editar

- **Número de WhatsApp:** busca `WA_NUMBER` en `index.html`.
- **Textos y fotos:** están directamente en `index.html` y en `assets/`.
